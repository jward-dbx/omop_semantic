# OMOP Semantic

This project contains OMOP (Observational Medical Outcomes Partnership) Common Data Model artifacts and tools for semantic analysis.

## Workspaces

### Vending Machine Workspace
- **Alias**: `vending-machine`
- **URL**: https://fe-sandbox-serverless-rde85f.cloud.databricks.com
- **Purpose**: Sandbox serverless workspace for OMOP semantic development and deployment

## MCP Servers

This project is configured with Databricks Managed MCP servers for AI-assisted development:
- `vending-machine-dbsql` - SQL query execution
- `vending-machine-system-ai` - Python execution and AI functions

## Getting Started

Configuration files are managed locally and not committed to git for security purposes.
