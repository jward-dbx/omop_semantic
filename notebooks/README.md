# Databricks Notebooks

This directory contains Databricks notebooks for interactive development and execution.

## Structure

- **exploration/**: Exploratory Data Analysis (EDA) notebooks
  - Data profiling
  - Initial data investigation
  - Proof of concepts
  
- **etl/**: ETL workflow notebooks
  - End-to-end ETL pipelines
  - Data transformation notebooks
  - Orchestration notebooks
  
- **analysis/**: Analytical notebooks
  - Healthcare analytics
  - Clinical insights
  - Reporting and visualization

## Naming Conventions

- Use descriptive names with context
- Include version or date if iterating: `person_eda_v2.py`
- Use Python (`.py`) format for version control

## Best Practices

1. Keep notebooks focused on a single task
2. Document cells with markdown
3. Clean outputs before committing
4. Use parameterized notebooks for reusability
5. Link to related documentation
